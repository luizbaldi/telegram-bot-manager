# Telegram Bot Manager
Simple Telegram bot manager, with a user interface built in ReactJS for frontend and NodeJS for backend.

## Setting up the environment
To prepare your project:

Frontend:
run
```sh
cd frontend/
npm install
```

Backend:
run
```sh
cd backend/
npm install
```
